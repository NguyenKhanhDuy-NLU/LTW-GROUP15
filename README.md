# Website đặt phòng khách sạn
Đây là  một website đặt phòng khách sạn, được xây dựng hoàn toàn bằng HTML, CSS và JavaScript.
Là một nền tảng web cung cấp giải pháp tìm kiếm và đặt phòng lưu trú trực tuyến.
Hệ thống bao gồm các luồng chức năng chính: Tìm kiếm địa điểm, Xem danh sách phòng, Xem chi tiết tiện ích và Giả lập quy trình đặt phòng.
